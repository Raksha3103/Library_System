# Library System

This project contains both frontend and backend for a Library Management System.

## Technologies Used
### Backend
- .NET Core
- C#
- Entity Framework
- SQL Server

### Frontend
- React JS
- Redux
- Jest (for testing)

## Project Structure
| Folder | Description |
| Library Management System | Backend API project (.NET) |
| LibraryTest | Unit test project |
| library_frontend | React frontend |

## How to Run the Project

### Backend Setup
1. Open `Library Management System` in Visual Studio
2. Update DB connection string in `appsettings.json`
3. Run migrations and start API

### Frontend Setup
```bash
cd library_frontend
npm install
npm start
